# Support

Thanks for using this project. This file explains where to get help and what information to include.

## Where to ask questions

Preferred channels:

- GitHub Discussions: <https://github.com/jscraik/.github/discussions>
- Documentation: Check the specific repository's README and docs folder

## Where to report bugs

Use the repository's **Bug report** issue form (Issues → New Issue → Bug report).

Please include:

- Version/commit SHA
- Steps to reproduce (minimal)
- Expected vs actual behavior
- Logs and environment details

## Feature requests

Use the repository's **Feature request** issue form.

## Security issues

**Do not report vulnerabilities here.** See `SECURITY.md`.

## Response expectations

This is an open source project. Support is best-effort and response times vary.
